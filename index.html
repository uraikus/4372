4372 Dashbaord Instructions

This application allows you to view all DAF 4372s in a single dashboard and modify them from the dashboard as well.

1. Download this HTML DAF 4372.
NOTE: If you are using Microsoft Teams, you will want to open the directory with everyone's 4372s and click "Sync," then follow the instructions from File Explorer.
2. Copy the downloaded "DAF 4372.html" into each member's folder on your computer/network drive.
3. Edit each DAF 4372 with the "Year" and the "Controller Name", otherwise the Dashboard will not load their records.
4. Download the 4372 Dashboard and save it wherever.
5. Open the Downloaded Dashboard and click "Open" button and select the parent directory of all your 4372s (it will scan subdirectories as well).
6. Change the year to that of the 4372s, and will will populate with each individual of that year with every month's entries.
NOTE: You can edit entries for a particular person of a particular month by clicking on the corresponding cell. It will physically update the 4372 as well as the tracker with the data.
Another NOTE: If someone's record isn't showing, ensure that the year and name are on their 4372s.
